# 导航器

脚手架通过监听 url 中的 query string `_route_` 的变化，实现页面之间的导航

import { Playground, Props } from 'docz'
import { Link, onNavigate, changeRouteKey } from '@deer-ui/admin-scaffold/router-multiple'

```js
import { Link, onNavigate, changeRouteKey } from '@deer-ui/admin-scaffold/router-multiple'

// changeRouteKey 用于改变监听的路由标识符
```

## onNavigate

onNavigate 为底层导航接口

```js
const navigatorConfig = {
  type: 'PUSH || GO_BACK || LINK',
  route: 'route',
}
onNavigate(navigatorConfig)
```
